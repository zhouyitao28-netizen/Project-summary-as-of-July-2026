### 前言

本项目基于MFE5340课程《AI in FE: Quantitative Investment》的Bootstrap方法论，针对A股市场118个常见因子的风险溢价进行统计推断研究。

传统金融统计中，我们常假设因子收益率服从正态分布或t分布来构建置信区间。然而，金融时间序列普遍存在自相关性（Autocorrelation）和厚尾特征，传统参数方法可能产生误导性的推断结果。

为此，本项目采用**Block Bootstrap（块自助法）**这一非参数统计方法，在保留时间序列依赖结构的前提下，对因子风险溢价的显著性进行稳健检验。研究对比了三种主流Block Bootstrap方法：
- **Stationary Bootstrap (SB)**：随机几何块长度，更好地保持平稳性
- **Circular Block Bootstrap (CBB)**：固定块长度，循环首尾避免边缘欠采样
- **Moving Block Bootstrap (MBB)**：经典固定块长度方法

### 对象描述

本研究覆盖118个常见量化因子，涵盖以下类别：
- **规模因子**：total_mv（总市值）、amount_21（21日成交额）等
- **动量因子**：idios_momentum_capm_252_21（特质动量）等
- **价值因子**：sellexp_to_total_mktcap_ttm（销售额市值比）等
- **技术因子**：tpi_21（交易价格指标）、abnormal_turnover_21（异常换手率）等

核心研究目标：对每个因子的**风险溢价均值**、**夏普比率**、**t统计量**构建95%置信区间，检验因子溢价是否显著区别于零。

### 代码思路
- 面向118个因子的风险溢价推断任务，对2004-2024年237个月度收益率数据，实现Stationary/Circular/Moving三种Block Bootstrap方法，采用块长度12个月、1000次重复，NumPy向量化计算均值、夏普比率、t统计量的95%置信区间。
- 执行三维度比较程序，发现MBB在分割样本覆盖概率上最优（84.7%），对比其在AR(1)蒙特卡洛模拟中真实覆盖率（91.8%接近95%）。说明Block Bootstrap方法无缺陷，整体覆盖率偏低来源于因子溢价随市场结构变化存在结构性漂移。
- 基于综合评分推荐MBB为最优方法，识别6个突出因子（total_mv、amount_21等），输出完整CI结果表及8套可视化图表，为因子择时研究提供实证依据。

### 结果

**1. 三种方法综合比较**

| 维度 | 指标 | SB | CBB | MBB | 最优方法 |
|------|------|-----|-----|-----|----------|
| **区间宽度** | Mean CI | 0.0130 | 0.0130 | 0.0130 | 持平 |
| | Sharpe CI | 0.2392 | 0.2388 | 0.2364 | MBB |
| | t-stat CI | 3.6831 | 3.6758 | 3.6397 | MBB |
| **自相关保持** | AR(1)偏差 | -0.0052 | -0.0022 | -0.0007 | MBB |
| | AR(1) RMSE | 0.0625 | 0.0666 | 0.0660 | SB |
| | AR(1)覆盖率 | 100% | 100% | 100% | 持平 |
| **覆盖概率** | Mean | 76.27% | 78.81% | 77.12% | CBB |
| (分割样本) | Sharpe | 73.73% | 74.58% | 72.03% | CBB |
| | t-stat | 79.66% | 81.36% | 83.05% | MBB |

**综合排名**（CI、偏差越小越好，覆盖概率、t统计量越大越好）：MBB推荐作为首选方法

**2. 覆盖概率偏低现象（72-85% vs 理论95%）**
- 所有三种方法的分割样本覆盖概率均显著低于名义水平95%
- **原因分析**：2020-2024年A股市场环境与2004-2019年存在结构性差异
这一点我们用蒙特卡洛模拟的反证
- 在AR(1)模拟数据下（平稳DGP），三种方法的覆盖概率均达到**92%左右**
  - SB: 92.0%
  - CBB: 92.2%
  - MBB: 92.7%
- **重要结论**：这证明代码实现无误，真实数据上的覆盖率偏低源于金融市场的**非平稳性**

**5. 研究意义**

- **传统参数检验的局限性**：假设平稳性和正态性可能导致过度自信的推断
- **因子投资的现实挑战**：即使历史表现优异的因子，在未来也可能因市场环境变化而失效
